# Knead To Know Website Design Asset Package

This ZIP package contains the organized branded design assets for **Knead To Know**, a boutique home bakery established in **2026** and located on **Daniel Island, South Carolina**.

The asset system is designed for a premium, coastal, local, handcrafted, bright-white bakery website. The visual direction uses the Knead To Know blue, white, and black palette with refined typography, generous whitespace, and minimal ornamental details.

These files are prepared for the Knead To Know website rebuild and are organized so they can be dropped into a website project without naming confusion.

## Intended Use

Use these assets while rebuilding the Knead To Know Bakery website from the stripped-down Spilled Milk website skeleton. The Spilled Milk website should only provide structural starting code or layout reference. All public-facing brand identity, copy, colors, product references, metadata, imagery, and UI components must become Knead To Know-specific.
