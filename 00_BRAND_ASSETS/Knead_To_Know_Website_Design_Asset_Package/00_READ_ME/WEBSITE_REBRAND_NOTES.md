# Website Rebrand Notes

The Spilled Milk website will be stripped down to its base structure and reused only as a layout/code skeleton.

The final website must be fully rebranded for Knead To Know Bakery.

## Remove or Replace

- Spilled Milk logo
- Spilled Milk colors
- Spilled Milk copy
- Spilled Milk product references
- Spilled Milk imagery
- Any unrelated branding
- Any old metadata, page titles, alt text, or SEO references

## Replace With

- Knead To Know logo
- Knead To Know blue/white/black brand palette
- Home bakery copy
- Daniel Island, South Carolina location references
- Artisan bread, cookies, small-batch bakery messaging
- Knead To Know website assets from this ZIP package

## Final Standard

The completed website should not look like a lightly edited Spilled Milk site. It should look like a custom, polished, boutique Knead To Know Bakery website built on a clean code skeleton.
