## Implementation Guidance

- Use PNG assets directly for high-fidelity visual sections, badges, and branded decorative elements.
- For interactive website components such as live buttons and forms, use these PNGs as visual references and recreate the styling in CSS where possible.
- Keep all generated PNG file names unchanged once added to the website project.
- Compress PNGs for production if file sizes impact load time.
- Add descriptive alt text when images are rendered as content.
- Use empty alt text for purely decorative dividers or motifs.
