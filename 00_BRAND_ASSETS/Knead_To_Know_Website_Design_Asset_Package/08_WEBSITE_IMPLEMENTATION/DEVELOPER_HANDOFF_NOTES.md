# Developer Handoff Notes

## Core Direction

Use the assets in this package to rebuild the website around the Knead To Know Bakery identity. The site should be bright white, refined, coastal, boutique, and professional.

## Must Do

- Use assets from this package.
- Preserve brand consistency across buttons, cards, forms, icons, banners, and badges.
- Keep the site bright white with blue and black accents.
- Strip Spilled Milk branding completely.
- Rename metadata and SEO fields.
- Check every image alt tag.
- Compress PNGs if needed for production performance.
- Maintain responsive behavior across desktop, tablet, and mobile.
- Use Daniel Island, South Carolina references naturally in copy and SEO.

## Must Not Do

- Do not distort logos or UI elements.
- Do not recolor assets outside the approved palette.
- Do not keep old Spilled Milk content or imagery.
- Do not use cartoon cupcake graphics or generic bakery clip art.
- Do not add loud colors, heavy textures, or cluttered backgrounds.
- Do not make the site feel rustic, template-based, or visually noisy.

## Recommended Build Standard

The final website should feel like a custom boutique bakery site, not a re-skinned template. Use the Spilled Milk code only for structure, then replace the visual layer, content layer, and metadata layer fully.
