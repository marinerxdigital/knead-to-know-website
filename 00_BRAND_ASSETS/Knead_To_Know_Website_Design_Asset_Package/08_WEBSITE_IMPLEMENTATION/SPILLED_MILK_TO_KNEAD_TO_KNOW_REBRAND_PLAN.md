# Spilled Milk to Knead To Know Rebrand Plan

## Objective

Use the existing Spilled Milk website only as a structural/code skeleton. Rebuild all visible brand identity, copy, visuals, product references, SEO metadata, and UI styling into a custom website for **Knead To Know Bakery**.

## What to Keep From Spilled Milk

- Clean layout structure if useful
- Responsive grid systems
- Navigation structure if it can be simplified
- Product-card logic or component shells
- Form/component scaffolding
- Existing routing only if it fits the new sitemap
- Basic ecommerce or ordering structure if present

## What to Remove

- Spilled Milk logo and brand marks
- Spilled Milk color palette
- Spilled Milk product names and descriptions
- Old photography and unrelated visuals
- Any old page titles, metadata, schema, or alt tags
- Any copy that references another business
- Any unused components that create visual clutter

## What to Replace

- Replace logo with `Knead_To_Know_Primary_Circular_Logo.png`.
- Replace CTAs with Knead To Know primary, secondary, and outline button assets or matching CSS components.
- Replace product cards with Knead To Know product-card styling.
- Replace testimonial blocks with Knead To Know testimonial-card styling.
- Replace form UI with Knead To Know input-field styling.
- Replace icons with the individual Knead To Know icon set.
- Replace old announcement bars with the Knead To Know promo banner strip.

## Asset Mapping

| Website Area            | Asset Direction                                                          |

| Header                  | Primary circular logo, clean nav, white background                       |

## Page-by-Page Rebrand Checklist

### Home

- Add Knead To Know logo.
- Use bright white background.
- Add hero copy focused on artisan bread, cookies, small-batch quality, and Daniel Island.
- Use primary and secondary CTAs.
- Add featured products, why choose us, testimonials, and contact/pre-order sections.

### Menu / Products

- Use product-card styling.
- Organize items by bread, cookies, seasonal goods, and custom orders.
- Use badges for Best Seller, Fresh Today, Seasonal, Pre-Order, and Small Batch.

### About

- Position the business as a boutique home bakery established in 2026 on Daniel Island.
- Emphasize handcrafted quality, trust, local pickup, and small-batch production.

### Contact / Custom Orders

- Use the form input field styling.
- Add inquiry fields: name, email, phone, order type, pickup date, message.
- Include contact envelope and location pin icons.

## SEO Checklist

- Replace all Spilled Milk page titles.
- Replace all metadata descriptions.
- Update Open Graph images and titles.
- Update alt text for every image.
- Add Daniel Island, South Carolina references where natural.
- Add local bakery phrases: home bakery, artisan bread, cookies, small-batch bakery, Daniel Island bakery.
- Check schema markup for business name, location, and product references.

## Final QA Checklist

- No Spilled Milk logo remains.
- No Spilled Milk text remains.
- No Spilled Milk colors remain.
- No unrelated imagery remains.
- All images have correct alt text.
- Header, footer, nav, and CTAs are responsive.
- Product cards work on desktop and mobile.
- Form fields are readable and accessible.
- CTAs have strong contrast.
- Website feels bright white, premium, coastal, and boutique.
